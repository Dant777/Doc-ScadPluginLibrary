ScadPluginLibrary

.NET Библиотека пользовательских расширений для SCAD++

Библиотека доступна в NuGet Packages.
Данная библиотека является оберткой над объектами и функциями “JScript пользовательские расширения” предусмотренные в SCAD++. Библиотека .NET повторяет весь функционал JScript расширений описанный в документации “Пользовательские расширения” поставляемый с программой SCAD++ не внося изменений в них.

Основная идея .NET библиотеки состоит в создании COM .dll плагина c последующей регистрацией в реестре Windows и вызов функционала .NET плагина стандартным способом JScript расширений через ActiveXObject, таким образом JScript расширения являются точкой входа для созданной .dll плагина.
