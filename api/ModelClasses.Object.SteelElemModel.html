SteelMark

Марка стали.

Declaration
public string SteelMark { get; set; }
Property Value
Type Description
System.String
Remarks

Параметр SteelMark зависит от выбранных норм проектирования.Допустимы следующие значения: "C235", "C245", "C255", "C275", "C285", "C345", "C345K", "C375", "C390", "C390K", "C440", "C590", "C590K", "20". При использовании ДБН В.2.6-163:2010 и ДБН В.2.6-198:2014 дополнительно можно использовать "C355". Отметим, что следует использовать символы "C" и "K" латинского алфавита.Если строка SteelMark является пустой, то будет использовано значение расчетного сопротивления, заданное параметром Ry.
